Un site web c'est quoi ?

Un site web c'est un ensemble de fichiers - codé en HTML - accesible depuis une adresse web. Il peut être : statique, c'est-à-dire qu'il possède un contenu fixe - exemple un roman, un article - ou dynamique, exemple d'un progiciel de gestion ou une eboutique. La différence étant une intéractivité avec l'utilisateur.

Technologie à connaître :

-Site Statique HTML

-Site dynamique HTML + PHP

CSS3

CSS est l'acronyme de "Cascading Style Sheets" qu'on pourrait traduire par "feuilles de style en cascade". Pour faire rapide CSS permet de gérer tout le style, le visuel d'une page web. Bordure, couleur, placement d'un élément, opacité, police d'écriture, marges, style de police, etc.Il agit sur des éléments HTML.
